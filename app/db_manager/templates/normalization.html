This section allow you to normalize files. It is important to add files to the database with the same normalization method. You can input files by batch and they will be returned as a zip file normalized. These are the files you should use to upload to the database. If your files are already normalized you can upload them directly. Your files will have the same name as the ones you input, however you must take into consideration that absolute intensity will not be shown in the new files.

Normalization is performed using the highest peak as 100% and calculating the other relative intensities as a percentage of this peak.
