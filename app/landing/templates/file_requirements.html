This program let you add txt files containing spectrum to ForeST. For the database and the consensus calculations to work correctly files must fill some requirements:

  • They must be text files (txt)
  • They should not contain more than two lines of text before the spectrum data
  • Files should follow the file name conventions GenusSpecie_wd_origin.txt:
    • No spaces
    • Underscores between fields
    • Dashes for amendments to specific fields
    • No periods or non-alphanumeric symbols
  • Files must contain mz as first column and normalized to base peak intensity as second colum. A third column with absolute intensity information will be disregarded by the program but it will remain in the original file. If your files only contain absolute intensity please normalize them before adding them to the database.
